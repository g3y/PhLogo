## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/g3y/PhLogo.git
   ```
2. Install NPM packages
   ```sh
   npm i node-html-to-image
   ```
3. Rename `Exemple` to what ever you wan't (in test.js)
